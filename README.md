# Bukalapak-pre-test

**API Test**

- go to `api-web-cypress`
- `yarn install`
- open cypress runner with `yarn cypress:open`
- choose `api-test.spec.ts`
- OR run `yarn cypress: run`

**Website Test**
- go to `api-web-cypress`
- `yarn install`
- open cypress runner with `yarn cypress:open`
- choose `register.spec.ts`
- OR run `yarn cypress: run`
